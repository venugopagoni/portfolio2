An Efficient Privacy-Preserving Credit Score System Based on Non interactive Zero-Knowledge Proof

Problem Statement

The credit system significantly impacts individuals, especially those with poor credit histories, while often neglecting privacy protection, risking sensitive user information leakage.

Approach

This project aims to develop a CSC model that quantifies credit risk while incorporating robust privacy protection measures through encryption and data anonymization techniques.

Result

The anticipated outcome is a secure credit assessment framework that enhances trustworthiness and protects user privacy, benefiting both individuals and financial institutions.

A Block chain based autonomus decentralized online social network.

Problem Statement

Current online social networks face issues of centralization, leading to privacy concerns and data security risks for users.

Approach

This project develops a decentralized OSN service using blockchain technology, providing secure peer-to-peer interactions with unique identities and private keys, utilizing Java for the front end and MySQL for the back end.

Result

The outcome will be a secure, trustworthy platform that enhances user privacy and transparency, mitigating data exploitation risks in online social interactions.
